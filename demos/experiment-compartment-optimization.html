Experiment / Batch Run Example of Fitting A Compartmental Model to Data

  • This is Succeptible, Exposed, Infected, Hospitalized, Funeral, and Removed model for Ebola in Liberia adapted from CM Rivers, 2014.
  • Plausible ranges for all parameters are defined, and a genetic algorithm is used to fit the model to the real Ebola case count data.
  • The genetic algorithm mutates parameter combinations by random mutation (towards the best fitting, using the gradient), and mates the top ten percent of combinations
