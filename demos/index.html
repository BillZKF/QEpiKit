QEpiKit - Javascript / Typescript Epidemiology Library

What is it?

This library provides broad access to tools for compartmental / differential equation (DE) and agent based (AB) modeling of communicable (and noncommunicable!) disease. Although developed with the public health / epidemiology / biomedical domains in mind, the tools themselves are domain neutral, and may be useful in other areas.

Why?

When I started development (while trying to learn more about epidemiological modeling), I could not find a free, open javascript library covering multiple approaches to epidemiological modeling. I chose Typescript / Javascript as the main language because they: 1) enable browser based examples on most devices 2) can be deployed to a server 3) will allow migration to ES6. For the AB portion of the library, I included methods from other fields including robotics and system dynamics.
